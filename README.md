# Rhetorical Trace

Rhetorical Trace is a computational tool for analyzing political speech through combined audio and linguistic features.

## Features

- Speech energy analysis (RMS over time)
- Pause detection and timing
- 5-second segment analysis
- Rhetorical marker tracking:
  - We/Us framing
  - Nationalist language
  - Crisis language
  - Antagonism detection
- Linguistic analysis:
  - Lexical diversity
  - Repetition
  - Modality
  - Pronoun density
  - Sentence structure

## Research Purpose

This tool is part of a doctoral research project in rhetoric and composition, focusing on how political language constructs identity, antagonism, and affect across different national contexts.

Initial case studies include:
- Donald Trump (U.S.)
- Giorgia Meloni (Italy)

## Technologies

- Python
- Streamlit
- Librosa
- Pandas
- Matplotlib
